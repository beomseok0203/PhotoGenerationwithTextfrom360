## Prerequisites

## Getting Started
### Installation
1. Install Caffe supporting RNN layers 
   -Run ./external/download_caffe.sh
   -Build caffe and pycaffe (http://caffe.berkeleyvision.org/installation.html) 
2. Run ./360_demo/preprosess.sh 
3. Run 360demo.py
